### How to run:
### Install Python 3.9.19
```
python3 --version
```
### Install PyQt6
```
pip install PyQt6
```
### Navigate to main directory
```
cd medical_clinic_system_app
```
### Run the GUI
```
python3 -m clinic gui
```
